配置 django-rq 在 admin 后台显示队列管理页面

周五 25 九月 2015

django Tags django-rq

效果：

详细步骤如下：

  1. 配置 settings.py

    RQ_SHOW_ADMIN_LINK = True

  2. 配置 urls.py

    urlpatterns += patterns(
    '',
    (r'^admin/django-rq/', include('django_rq.urls')),
)

  3. Done!

Demo 项目： django_rq_admin
