让我们一起来阅读源代码之序

2018-03-04

python Tags let-us / let-us-read-source-code

准备开新坑：让我们一起来阅读源代码。准备写阅读多个项目的源代码的文章。

计划¶

初步计划阅读的项目如下（参考了 Reading Great Code , 看情况调整，不一定就是下面的这些）：

  • Howdoi
  • Tablib
  • Requests
  • Bottle
  • Flask
  • Werkzeug

约定¶

同时也约定一些对文章内容的要求：

  • 新手友好，尽量写的能够让新手也能看懂。
  • 不是单纯的项目代码说明，而是会从新手的角度讲讲如何开始看一个项目的源代码， 从哪里入手，为何从那里入手（比如有一些约定俗成的东西 或者有一些是工具和框架的用法让我们确定入口在那里）。
  • 对于项目中的一些比较高级的知识点会先单独写文章进行介绍，然后再写源码阅读的文章。
  • 为了保持连贯性，文章会几篇几篇的发布，一篇一篇的话可能会因为间隔的原因看了就忘。
  • 前期应该都是一些知名的 Python 项目，后续可能也会有一些不知名的小项目， 也有可能会有其他语言的项目。
