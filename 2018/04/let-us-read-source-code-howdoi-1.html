让我们一起来阅读源代码之 Howdoi （一）

2018-04-07

python Tags let-us / let-us-read-source-code / Howdoi

前言¶

正式开始填坑了。让我们先从 Howdoi 这个项目开始。有一点要说明一下：我使用的 Python 版本是 Python 3.6 ，我的源码阅读的所有文章中所有的 Python 相关的操作如果没有特别说明都是用的 Python 3.6 。

熟悉基本功能¶

既然要看它的源代码至少先得了解程序的基本功能，这样看代码的时候才会有“哦，原来那个功能是这样实现的呀”的感觉。熟悉项目的基本功能有哪些是首先要做的事情。

我们先通过阅读项目的 README 来了解这个项目的功能。通过 README 我们可以知道 Howdoi 这个项目的主要功能如下：

  • 用来在命令行中搜索编程问题，输出答案代码
  • 默认使用 google 搜索 StackOverflow 上的问题，可以通过环境变量 HOWDOI_SEARCH_ENGINE 修改搜索引擎改为使用 bing 搜索。
  • 可以通过环境变量 HOWDOI_URL 指定搜索不同的问答网站（默认是 stackoverflow.com ，可以是其他 stackoverflow 家族的网站，比如：serverfault.com, pt.stackoverflow.com）。

下面我们来实际使用一下这个程序，看看实际的效果。

安装¶

git clone 源代码（为了保持一致，使用我 fork 的代码）:

$ git clone https://github.com/let-us-read-source-code/howdoi.git
$ cd howdoi

安装程序：

  1. 我们先使用 virtualenv 创建虚拟环境，防止污染系统 python 环境

    $ python -m venv venv
$ . venv/bin/activate

  2. 使用 develop 模式安装，方便调试（修改代码后不需要重新安装，修改会直接生效）:

    $ pip install -e .
$ howdoi

使用¶

为了方便，我们通过环境变量 HOWDOI_SEARCH_ENGINE 指定使用 bing 搜索引擎:

$ export HOWDOI_SEARCH_ENGINE=bing

先来看看最基本的搜索问题的功能:

$ howdoi python convert int to str
>>> str(10)
'10'
>>> int('10')
10

效果还是可以的。

再来看一下几个主要参数的功能:

-p POS, --pos POS     选择指定位置的答案 (default: 1):

$ howdoi -p 1 python convert int to str
>>> str(10)
'10'
>>> int('10')
10

$ howdoi -p 2 python convert int to str
>>> a = "545.2222"
>>> float(a)
545.22220000000004
>>> int(float(a))
545

-a, --all             显示答案全文:

$ howdoi -a python convert int to str
★  Answer from https://stackoverflow.com/questions/961632/converting-integer-to-string-in-python ★
>>> str(10)
'10'
>>> int('10')
10

Links to the documentation:

The problem seems to come from this line: d.str().
Conversion to a string is done with the builtin str() function, which basically calls the __str__() method of its parameter.

-l, --link            只显示答案链接:

$ howdoi -l python convert int to str
https://stackoverflow.com/questions/961632/converting-integer-to-string-in-python

-n NUM_ANSWERS, --num-answers NUM_ANSWERS 返回的答案数目:

$ howdoi -n 2 python convert int to str
★  Answer from https://stackoverflow.com/questions/961632/converting-integer-to-string-in-python ★

熟悉了基本功能后，我们即将进入阅读源代码的环节，不过鉴于这篇文章已经有点长了，这次就先到这里。

总结¶

本文我们主要熟悉了一下 howdoi 这个项目的基本功能，为后面的阅读源码工作做一点准备， 后面我们将进入涉及源代码的环节。在下篇文章出炉前大家不妨思考一下：如果让你来实现 howdoi 的这几个基本功能，你会如何实现？
