Markdown

多级列表

官方的 Markdown 语法：

* list
  * sublist
  * sublist
* list

pelican 使用的是 python-markdown 来解析 markdown 文件， 而 python-markdown 的关于多级列表的语法跟官方的有点不一样：

* list
    * sublist
    * sublist
* list

子列表与父列表之间的对齐方式是：至少空出4个空格或1一个 Tab 键。 官方的语法兼容 python-markdown 的语法，所以不要担心兼容性。
