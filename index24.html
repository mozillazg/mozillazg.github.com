[django]解决发送密码重置邮件的发件人是 "webmaster@localhost" 的问题

Date 周二 04 六月 2013 Category django Tags django / python / send_mail

默认情况下，django 将通过 webmaster@localhost 发送密码重置邮件。

这明显不是我们想要的，更改 settings.py 文件即可修复这个问题。

增加一个配置项 DEFAULT_FROM_EMAIL：

DEFAULT_FROM_EMAIL = 'foo@bar.com'

此时的发件人将会是：foo@bar.com

[javascript]列出对象所有属性

Date 周四 23 五月 2013 Category web Tags javascript / web

当我们想知道某个对象拥有哪些的属性时，可以用下面的方法（现代浏览器）：

var properties = Object.keys(obj);

这个方法对于我们使用第三方 javascript 插件，而该插件的文档不是很详细时有很大的帮助

trackFormatter: function(obj){
  console.debug('properties:');
  console.debug(Object.keys(obj));
}

[linux]screen 常用命令

Date 周四 16 五月 2013 Category linux Tags linux / screen

通过 screen 命令可以让终端命令在我们断开远程服务器后依旧运行而不会中断。

安装 screen 命令

redhat/centos:

yum install screen

ubuntu:

sudo apt-get install screen

常用的 screen 命令

新建一个名为 django 的 screen shell：

screen -S django

新建一个名为 django 的后台 screen shell：

screen -dS django

查看所有的 screen shell:

$ screen -ls
