[python]使用 multiprocessing.dummy 执行多线程任务

Date Sun 19 January 2014 Category python Tags python / multiprocessing / thread / threading

本文将简单讲述一种使用 multiprocessing.dummy 模块执行多线程任务的方法。

[python]输出一个简单的进度条

Date Mon 13 January 2014 Category python Tags python

使用 sys.stdout 或 sys.stderr 就可以输出一个简单的进度条：

import time
import sys

f = sys.stdout

for n in range(1, 101):
    s = ('#' * n).ljust(100, '-')
    f.write(s)
    f.flush()
    time.sleep(0.3)
    f.write('\r')
f.write('\n')
