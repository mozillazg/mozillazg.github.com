[linux]配置 vsftp 启用被动模式（passive mode）解决客户端"no route to host"错误

Date Tue 20 May 2014 Category linux Tags linux / vsftp

系统: CentOS

  • 修改 vsftpd.conf，启用被动模式，指定端口:

    pasv_min_port=12000
pasv_max_port=12199
pasv_enable=YES

  • 配置防火墙 iptables，允许端口访问:

    iptables -I INPUT -p tcp --dport 12000:12199 -j ACCEPT
service iptables save

[linux] CentOS 下安装 percona-toolkit

Date Mon 24 March 2014 Category database Tags centos / mysql

percona-toolkit 是一组高级命令行工具的集合， 用来执行各种复杂和麻烦的 MySQL 操作。

下面记录一下在 CentOS 下如何通过 rpm 安装 percona-toolkit:

  1. 安装依赖
    yum install perl perl-IO-Socket-SSL perl-Time-HiRes

  2. 下载 rpm 包
    wget percona.com/get/percona-toolkit.rpm

  3. 安装 rpm 包
    rpm -ivh percona-toolkit-*.rpm
