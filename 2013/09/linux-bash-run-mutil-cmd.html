[linux]bash 下 ; && || 的区别

周六 28 九月 2013

linux Tags linux / bash / shell

linux 中 bash 下执行多个命令时，; && || 操作符的区别：

cmd1 ; cmd2 cmd1 和 cmd2 都会 被执行
cmd1 && cmd2 如果 cmd1 执行 成功 则执行 cmd2
cmd1 || cmd2 如果 cmd1 执行 失败 则执行 cmd2
