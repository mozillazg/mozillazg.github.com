[python] 发送邮件时设置邮件的优先级/重要性

Date 周二 02 七月 2013 Category python Tags python / email / X-Priority

设置邮件 header 中 X-Priority 的值就可以了：

X-Priority 的值 含义
"1" 最高级别（重要性高）
"2" 介于中间
"3" 普通级别（不提示重要性）
"4" 介于中间
"5" 最低级别（重要性低）
