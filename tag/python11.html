[django]解决发送密码重置邮件的发件人是 "webmaster@localhost" 的问题

Date Tue 04 June 2013 Category django Tags django / python / send_mail

默认情况下，django 将通过 webmaster@localhost 发送密码重置邮件。

这明显不是我们想要的，更改 settings.py 文件即可修复这个问题。

增加一个配置项 DEFAULT_FROM_EMAIL：

DEFAULT_FROM_EMAIL = 'foo@bar.com'

此时的发件人将会是：foo@bar.com

[django]__unicode__: Bad Unicode data 问题

Date Sun 05 May 2013 Category django Tags django / python

问题：

>> models.Foo.objects.all()
>> [<Foo: [Bad Unicode data]>]

解决办法：

更改 model 的 __unicode__ 方法：

# Before
def __unicode__(self):
    return '%s' % self.get_foo_display()  # it's bad

# To
def __unicode__(self):
    return u'%s' % self.get_foo_display()  # it's Ok
