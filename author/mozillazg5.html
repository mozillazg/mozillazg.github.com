  2. [django]解决发送密码重置邮件的发件人是 "webmaster@localhost" 的问题

    周二 04 六月 2013
    By mozillazg

    In django.

    tags: djangopythonsend_mail

    默认情况下，django 将通过 webmaster@localhost 发送密码重置邮件。

    这明显不是我们想要的，更改 settings.py 文件即可修复这个问题。

    增加一个配置项 DEFAULT_FROM_EMAIL：

    DEFAULT_FROM_EMAIL = 'foo@bar.com'

    此时的发件人将会是：foo@bar.com

  3. [django]使用自定义 context processor 实现模板全局变量

    周三 23 一月 2013
    By mozillazg

    In django.

    tags: pythondjango

    本文使用的是 django 1.4.3。

    写一个 context processor 函数

    可以把这个函数放在项目的任何地方，本文是放在应用目录下的 context_processors.py 文件中：

    from hello.models import Category


def categories(request):  # 只有一个参数（HttpRequeset 对象）
    all_categories = Category.objects.all()
    context = {'categories': all_categories}

    return context  # 返回值必须是个字典

    注意：context processor 函数只有一个参数，即 HttpRequest 对象，并且函数的返回值必须是个字典。
