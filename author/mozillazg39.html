  • 用 nginx 做反向代理、负载均衡和静态文件服务器
  • 同时起多个 tornado 进程服务多个端口

假设配置文件如下:

upstream tornado_server {
    server 127.0.0.1:5001;

  1. 更新代码

  2. 使用不同的端口启动新的 tornado 服务: start tornado_5002
