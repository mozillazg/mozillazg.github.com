    周四 23 五月 2013
    By mozillazg

    In web.

    tags: javascriptweb

    当我们想知道某个对象拥有哪些的属性时，可以用下面的方法（现代浏览器）：

    var properties = Object.keys(obj);

    这个方法对于我们使用第三方 javascript 插件，而该插件的文档不是很详细时有很大的帮助

    trackFormatter: function(obj){
  console.debug('properties:');
  console.debug(Object.keys(obj));
}

  2. [linux]创建指定大小的文件

    周六 03 八月 2013
    By mozillazg

    In linux.

    tags: linux

    比如创建 10M 大小的文件：

    # 创建大小为 10M 的文件：abc.exe
dd if=/dev/zero of=abc.exe bs=1M count=10

# 只生成属性为 10M 大小的文件，但不占用实际硬盘空间
dd if=/dev/zero of=abc.exe bs=1M seek=10 count=0
